Cheat Sheet for HTML and CSS

This cheat sheet is developed and updated by Ankit Singh

Name of Tag/Element Symbol Description
doctype <!DOCTYPE html> initial declaration for browser to know that file type is html
html <html> start of html file
head <head> head of the html file. includes titles and links to css files.
link <link> contains link to html css file Syntax: <link href="style.css" rel="stylesheet">
title <title> Displays the title in tab
body <body> Contains the data displayed on webpage
header <header> Section/division for storing heading related data of the webpage
nav <nav> Sementic Section for displaying the navigation menu
ordered list <ol> Creates a list with numbering from 1-n
unordered list <ul> Creates a lsit without any numbering with only bullet points.
main <main> Section/division that stores the main displayed content of the webpage
article <article> Sementic html section for storing the core content of the webpage
figure <figure> section for displaying images
figure caption <figcaption> Caption for the image. Must be included inside figure section. So stays attached to the image.
audio/video <audio> <video> Embeding audio &video files. Syntax: <audio/video src="x.y">error Message</audio/video>
image <img> Displays the image. Syntax: <img src="x.y" alt=" ">
hyperlink <a> Creates a hyper link. Syntax: <a src="url" target="_blank">Name of HyperLink</a>
additional information <aside> Displays additional information that enhances a section but is not mandatory
title <title> Displays the title in tab
footer <footer> Section/division that stores the foot notes of the webpage
